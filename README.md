## Get started

Environment
- yarn (3.6.4)
- 

1. Install dependencies

   ```bash
   yarn install
   ```

2. Start the app

   With Expo
   ```bash
   yarn expo:ios
   yarn expo:android
   ```
   With Build
   ```bash
   yarn start (to start Metro)
   yarn ios
   yarn android
   ```
